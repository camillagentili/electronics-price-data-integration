# Electronics Price Data Integration

This repository contains a **university group project** developed for the **Data Management course**.

The project focuses on collecting, preprocessing, integrating and improving the quality of product price data from three major Italian electronics retailers: Euronics, Expert and Unieuro.

The goal of the project is to build a complete **data management pipeline**, starting from web scraping and ending with an integrated dataset suitable for analysis.

## Project Workflow

1. **Data Scraping**
   - Product data collected from retailer websites.

2. **Preprocessing**
   - Cleaning and standardizing product attributes.

3. **Data Integration**
   - Matching products across different retailers.

4. **Data Quality**
   - Duplicate detection and dataset validation.

5. **Final Dataset**
   - Creation of an integrated dataset containing products from all sources.
